{{- /* Weekly Frequency Handler Handles weekly recurrence patterns by extracting and centralizing the weekly frequency logic that was previously embedded in the monolithic recurrence_human_readable.html template. This is part of Phase 2 of the Hugo template refactoring project (RECUR-007). Input: dict "rule" $rule where $rule contains the recurrence rule data - rule.freq: "WEEKLY" (case-insensitive) - rule.byDay: optional, single day code ("MO") or array of day codes (["MO", "FR"]) - rule.interval: optional, defaults to 1 Output: String with weekly frequency description Pattern Priority (highest to lowest): 1. Weekly with specific days: "Every Monday", "Every Monday and Friday" 2. Weekly with interval and days: "Every 2 weeks on Monday" 3. Simple weekly with interval: "Every 2 weeks" 4. Simple weekly: "Every week" Acceptance Criteria Examples: - {freq: "WEEKLY", byDay: "MO"} → "Every Monday" - {freq: "WEEKLY", interval: 2} → "Every 2 weeks" - {freq: "WEEKLY", byDay: ["MO", "FR"], interval: 1} → "Every Monday and Friday" - {freq: "WEEKLY", byDay: "TU", interval: 3} → "Every 3 weeks on Tuesday" Phase 1 Utility Integration: - Uses interval_formatter.html for standardized interval formatting - Uses day_names_resolver.html for day code to name conversion Error Handling: - Validates required 'rule' parameter is present - Validates rule contains required frequency data - Provides meaningful error messages for debugging - Handles edge cases gracefully */ -}} {{- /* Input validation - ensure rule parameter is provided */ -}} {{- if not (isset . "rule") -}} {{- errorf "weekly_frequency.html: Missing required parameter 'rule'. Usage: {{ partial \"recurrence/weekly_frequency.html\" (dict \"rule\" $rule) }}" -}} {{- end -}} {{- $rule := .rule -}} {{- /* Validate that rule parameter is not empty */ -}} {{- if not $rule -}} {{- errorf "weekly_frequency.html: Parameter 'rule' cannot be empty or nil" -}} {{- end -}} {{- /* Extract and validate frequency */ -}} {{- $freq := $rule.freq | lower -}} {{- if ne $freq "weekly" -}} {{- errorf "weekly_frequency.html: Expected frequency 'weekly', got '%s'. This handler only processes weekly patterns." $freq -}} {{- end -}} {{- /* Extract rule parameters with case-insensitive support */ -}} {{- $byDay := or $rule.byDay $rule.byday -}} {{- $interval := $rule.interval | default 1 -}} {{- /* Validate interval is positive */ -}} {{- if le $interval 0 -}} {{- errorf "weekly_frequency.html: Interval must be a positive integer, got: %v" $interval -}} {{- end -}} {{- /* Initialize result variable */ -}} {{- $result := "" -}} {{- /* Pattern matching with priority-based logic: Priority 1: Weekly with specific days (byDay present) Priority 2: Simple weekly with interval (no byDay) */ -}} {{- if $byDay -}} {{- /* Priority 1: Weekly with specific days */ -}} {{- /* Use day_names_resolver utility to convert day codes to names */ -}} {{- $dayNames := partial "recurrence/day_names_resolver.html" (dict "days" $byDay) -}} {{- if eq $interval 1 -}} {{- /* Simple case: Every [day(s)] */ -}} {{- $result = printf "Every %s" $dayNames -}} {{- else -}} {{- /* Complex case: Every N weeks on [day(s)] */ -}} {{- /* Use interval_formatter utility for consistent interval formatting */ -}} {{- $intervalText := partial "recurrence/interval_formatter.html" (dict "interval" $interval "unit" "week") -}} {{- /* Replace "Every" with "on" to create proper grammar */ -}} {{- $intervalPart := $intervalText | replaceRE "^Every " "" -}} {{- $result = T "ical_recurrence_every_interval_on_days" (dict "interval" $intervalPart "days" $dayNames) -}} {{- end -}} {{- else -}} {{- /* Priority 2: Simple weekly with interval (no specific days) */ -}} {{- /* Use interval_formatter utility for consistent formatting */ -}} {{- $result = partial "recurrence/interval_formatter.html" (dict "interval" $interval "unit" "week") -}} {{- end -}} {{- /* Return the formatted result */ -}} {{- $result -}}
